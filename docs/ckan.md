작성중
{: .label .label-yellow }

[CKAN](https://ckan.org/)은 데이터 개방 표준을 주도하는 오픈소스 플랫폼이에요.
미국·캐나다·호주·스위스 등의 정부 공공데이터 플랫폼으로 채택하고 있고, 우리 나라 역시 여러 기관이 CKAN을 쓰고 있어요.

[기미나인](https://gimi9.com/)은 CKAN 2.9를 기반으로 하고 데이터 Curation을 위한 기능을 더했어요.
