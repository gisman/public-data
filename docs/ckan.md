작성중
{: .label .label-yellow }

[CKAN](https://ckan.org/)은 데이터 개방 표준을 주도하는 오픈소스 플랫폼입니다.
미국, 캐나다, 호주, 스위스 등의 정부는 공공데이터 플랫폼으로 CKAN을 채택하였습니다. 또한, 대한민국에서도 여러 기관이 CKAN을 사용하고 있습니다.

[기미나인](https://gimi9.com/)은 CKAN 2.9를 기반으로 하며, 데이터 Curation을 위한 추가 기능을 제공합니다.
